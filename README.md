# tabash

![banner](./src/assets/banner.png)

tabash, a bash-like 'new tab' page for browser.

## Setup

install

```Shell
yarn
```

start

```Shell
yarn start
```

## Examples

> type "help" in tabash for more infos

list registered links

```Shell
ls
```

register link

```Shell
gh=github.com
```

open link (github)

```Shell
gh
```
